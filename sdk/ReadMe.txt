«Android» — библиотека криптопровайдера под ОС Android с примером;

«iOS» — библиотека криптопровайдера под ОС iOS с примерами;

«CC (Kalkan)» — библиотеки криптопровайдера, основанная на исходном коде продукта OpenSSL.
Интерфейсы библиотеки практически полностью соответствует API фреймворка OpenSSL.
По этой причине директория содержит официальную документацию к OpenSSL.
В директории расположены сборки под ОС Linux и Windows, кроме того имеются обертки для платформы .Net
с базовым функционалом. Кроме того, имеются различные примеры для работы с библиотекой.

«Java(Kalkan)» — библиотека криптопровайдера для платформы Java и дополнительные необходимые библиотеки.
Кроме того в директории представлен Java апплет, который проставляет базовые функции подписания и проверки.
Представлены различные примеры и дкоументация.

«Keys and Certs» — директория содержащая тестовые ключи и сертификаты корневых УЦ. Директория «Текущий НУЦ (old oids)»
содержит сертификаты выданные реальным Удостоверяющим Центром, но содержащие недействительные данные (данные поля Субъект).
Директория «Новый НУЦ (new oids)» содержит сертификаты выданные НОВЫМ ТЕСТОВЫМ Удостоверяющим Центром.
Структура сертификатов, расположенных директории «Новый НУЦ (new oids)» аналогична структуре сертификатов,
которые будут выдаваться НОВЫМ БОЕВЫМ Удостоверяющим Центром в начале декабря 2015 года.

NCALayer — приложение для решения проблемы работы Java апплетов в браузерах (см. NPAPI и Google).
Данное решение позволяет использовать любой браузер и операционную систему, на которой установлена JRE. 
Минимальная версия JRE должна быть — 1.6. ReadMe к данному приложению обязателен для прочтения!
